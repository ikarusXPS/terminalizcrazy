# Tastenkürzel

> Vollständige Referenz aller Tastenbelegungen

## Übersicht

TerminalizCrazy verwendet intuitive Tastenkürzel für effiziente Navigation und Bedienung. Die meisten Kürzel folgen gängigen Terminal-Konventionen.

---

## Chat-Modus

### Texteingabe

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Enter` | Senden | Nachricht an KI senden |
| `↑` | History zurück | Vorherigen Befehl aus History laden |
| `↓` | History vorwärts | Nächsten Befehl aus History laden |
| `Page Up` | Scrollen | Chat-Verlauf nach oben scrollen |
| `Page Down` | Scrollen | Chat-Verlauf nach unten scrollen |

### Befehlsausführung

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Ctrl+E` | Ausführen | Letzten vorgeschlagenen Befehl ausführen |
| `Ctrl+R` | Anzeigen | Letzten Befehl im Chat anzeigen |
| `Ctrl+Y` | Kopieren | Befehl ins System-Clipboard kopieren |

### Session-Verwaltung

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Ctrl+L` | Leeren | Chat-Verlauf löschen (Session bleibt erhalten) |
| `Esc` | Beenden | Anwendung beenden |
| `Ctrl+C` | Abbrechen | Laufende Operation abbrechen / Anwendung beenden |

### KI-Steuerung

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Ctrl+A` | Agent-Modus | Agent-Modus umschalten (off → suggest → auto) |
| `Ctrl+M` | Modell-Auswahl | KI-Modell wechseln |

---

## Zusammenarbeit

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Ctrl+S` | Teilen | Session für Zusammenarbeit freigeben |
| `Ctrl+J` | Beitreten | Einer geteilten Session beitreten |
| `Ctrl+D` | Trennen | Zusammenarbeit beenden |

---

## Tabs

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Ctrl+T` | Neuer Tab | Neue Terminal-Session öffnen |
| `Ctrl+W` | Tab schließen | Aktuellen Tab schließen |
| `Ctrl+Tab` | Nächster Tab | Zum nächsten Tab wechseln |
| `Ctrl+Shift+Tab` | Vorheriger Tab | Zum vorherigen Tab wechseln |
| `Alt+1` bis `Alt+9` | Direktwahl | Direkt zu Tab 1-9 wechseln |

---

## Panels (Panes)

### Navigation

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Alt+←` | Links | Zum Panel links wechseln |
| `Alt+→` | Rechts | Zum Panel rechts wechseln |
| `Alt+↑` | Oben | Zum Panel oben wechseln |
| `Alt+↓` | Unten | Zum Panel unten wechseln |

### Verwaltung

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Ctrl+\` | Vertikal teilen | Panel vertikal teilen |
| `Ctrl+-` | Horizontal teilen | Panel horizontal teilen |
| `Ctrl+W` | Panel schließen | Aktuelles Panel schließen |
| `Ctrl+Z` | Zoom | Panel-Zoom umschalten |

---

## Bestätigungsdialoge

Bei Befehlen mit erhöhtem Risiko erscheint ein Bestätigungsdialog:

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Y` | Ja | Befehl ausführen |
| `N` | Nein | Befehl abbrechen |
| `Esc` | Abbrechen | Dialog schließen |

---

## Session-Auswahl (Startbildschirm)

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `↑` / `k` | Nach oben | Vorherige Session auswählen |
| `↓` / `j` | Nach unten | Nächste Session auswählen |
| `Enter` | Auswählen | Ausgewählte Session laden |
| `N` | Neue Session | Direkt neue Session starten |
| `Esc` | Neue Session | Auswahl überspringen, neue Session |

---

## Collaboration-Beitritt

| Taste | Aktion | Beschreibung |
|-------|--------|--------------|
| `Buchstaben/Zahlen` | Eingabe | Share-Code eingeben |
| `Enter` | Beitreten | Mit eingegebenem Code beitreten |
| `Backspace` | Löschen | Letztes Zeichen löschen |
| `Esc` | Abbrechen | Zurück zum Chat |

---

## Schnellreferenz

### Wichtigste Kürzel

```
Ctrl+E    Befehl ausführen
Ctrl+Y    Befehl kopieren
Ctrl+A    Agent-Modus umschalten
Ctrl+M    Modell-Auswahl
Ctrl+S    Session teilen
Ctrl+J    Session beitreten
Ctrl+L    Chat leeren
↑/↓       History durchsuchen
Esc       Beenden
```

### Panel-Navigation

```
Alt+Pfeile    Panel wechseln
Ctrl+\        Vertikal teilen
Ctrl+-        Horizontal teilen
Ctrl+Z        Zoom
Ctrl+W        Schließen
```

### Tab-Navigation

```
Ctrl+T            Neuer Tab
Ctrl+W            Tab schließen
Ctrl+Tab          Nächster Tab
Ctrl+Shift+Tab    Vorheriger Tab
Alt+1-9           Direkt zu Tab
```

---

## Anpassung

Tastenkürzel können derzeit nicht angepasst werden. Diese Funktion ist für zukünftige Versionen geplant.
